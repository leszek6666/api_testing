# api_testing

automation api endpoints

testing site is:
[link](https://airportgap.dev-tester.com/docs#api_ref_get_airports)

# To run test

1. install python and pip

`python --version`
`pip --version`

2. install request library

`pip install requests`

3. in cli run
   `pytest test.py`

   or

   `python3`
   and choose test case for example:
   `test_get_airports`
